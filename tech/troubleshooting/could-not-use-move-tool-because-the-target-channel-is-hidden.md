#Could not use move tool because the target channels is hidden

This article explains why while using Adobe Photoshop you might get “Could not use move tool because the target channel is hidden” warning and how to deal with it.

Why are you getting “Could not use move tool because the target channel is hidden”  popup warning?

If you get this error while trying to select object with Move Tool [V] it means that have entered “Edit in quick mask mode”.

If you are using keyboard shortcut it is that most likely you have accidentally hit [Q].

How to disable the popup?

Simply quit “Edit in quick mask” mode. You can do that by doing one of the following:

* Use keyboard shortcut, press [Q] again;
* Click “Select” from main menu and then un-tick “Edit in quick mask mode”;
* Select “Channels” panel and delete Alpha layer
